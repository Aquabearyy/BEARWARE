# Silent Hub

Simple and powerful hub for popular Roblox games.

## Supported Games
- Slap Battles 
- Slap Royale
- KAT
- Rivals

## Usage
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/sxlent404/SilentHub/main/Loader.lua"))()
```

## Features
- Clean interface
- Auto-updates
- Built-in security
- Game detection
